Project Features

  • Real-time step-by-step animations of algorithms
  • User-controlled execution with play, pause, and step controls
  • Input customization for testing different scenarios
  • Multi-platform support (desktop, mobile, tablets)
  • Self-explanatory interface with intuitive controls
  • Educational value for classroom demonstrations and self-study

Technical Architecture

The system follows a modular, object-oriented architecture built with vanilla JavaScript and HTML5 Canvas:

  • Presentation Layer: HTML pages, CSS styling, and UI controls
  • Algorithm Layer: Base algorithm classes and specific implementations
  • Animation Layer: Animation engine and animated objects
  • Rendering Layer: HTML5 Canvas 2D context and timing APIs

Available Visualizations

The project includes over 50 algorithm implementations covering:

  • Basic Data Structures: Stacks, Queues, Lists
  • Trees: BST, AVL, Red-Black, Splay Trees, B-Trees
  • Hash Tables: Open and Closed addressing
  • Sorting Algorithms: Comparison and non-comparison sorts
  • Graph Algorithms: BFS, DFS, Shortest Path, MST
  • Dynamic Programming: Fibonacci, Making Change, LCS
  • Heaps: Binary, Binomial, Fibonacci, Leftist
  • Other Algorithms: Recursion, Geometric, Disjoint Sets

How to Use

The visualizations are designed to be intuitive and educational. Here's how to get the most out of them:

Algorithm Controls

At the top of each visualization, you'll find algorithm-specific controls including input fields for data entry and action buttons for operations like Insert, Delete, and Find.

Animation Controls

  • Skip Back: Undo current command or go to previous command
  • Step Back: Move backward one frame (when paused)
  • Play/Pause: Toggle automatic playback and manual stepping
  • Step Forward: Move forward one frame (when paused)
  • Skip Forward: Jump to end of current animation
  • Speed Control: Adjust playback speed with slider
