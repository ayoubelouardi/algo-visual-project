About Algorithm Visualization Project

This project is a comprehensive collection of interactive visualizations for fundamental data structures and algorithms. Built with HTML5 Canvas and JavaScript, it provides an educational platform for understanding how various algorithms work through step-by-step animations.

Originally created by David Galles at the University of San Francisco, this project continues to serve as an excellent learning resource for computer science students and educators worldwide.

Project Features

  • Real-time step-by-step animations of algorithms
  • User-controlled execution with play, pause, and step controls
  • Input customization for testing different scenarios
  • Multi-platform support (desktop, mobile, tablets)
  • Self-explanatory interface with intuitive controls
  • Educational value for classroom demonstrations and self-study

Technical Architecture

The system follows a modular, object-oriented architecture built with vanilla JavaScript and HTML5 Canvas:
  • Presentation Layer: HTML pages, CSS styling, and UI controls
  • Algorithm Layer: Base algorithm classes and specific implementations
  • Animation Layer: Animation engine and animated objects
  • Rendering Layer: HTML5 Canvas 2D context and timing APIs

Available Visualizations

The project includes over 50 algorithm implementations covering:
  • Basic Data Structures: Stacks, Queues, Lists
  • Trees: BST, AVL, Red-Black, Splay Trees, B-Trees
  • Hash Tables: Open and Closed addressing
  • Sorting Algorithms: Comparison and non-comparison sorts
  • Graph Algorithms: BFS, DFS, Shortest Path, MST
  • Dynamic Programming: Fibonacci, Making Change, LCS
  • Heaps: Binary, Binomial, Fibonacci, Leftist
  • Other Algorithms: Recursion, Geometric, Disjoint Sets

How to Use the Visualizations

The visualizations are designed to be intuitive and educational. Here's how to get the most out of them:


Algorithm Specific Controls

At the top of each visualization page, you'll find algorithm-specific controls that change depending on what you're visualizing. These typically include:
  • Input fields for entering data (numbers, text, etc.)
  • Action buttons for operations like Insert, Delete, Find
  • Specialized controls for algorithm-specific features
Simply enter your data and click the appropriate button or press Enter to start the visualization.

General Animation Controls

  • Skip Back: Completely undo the current command or go back to the previous command
  • Step Back: Step backward one frame in the current animation (only active when paused)
  • Play/Pause: Toggle between automatic playback and manual stepping mode
  • Step Forward: Step forward one frame in the current animation (only active when paused)
  • Skip Forward: Jump to the end of the current animation
  • Animation Speed (slider): Adjust the playback speed (settings are saved in cookies)
  • Canvas Size (w, h): Modify the display area dimensions for different screen sizes
  • Move Controls: Toggle control position between top and bottom of the page

Tips for Learning

  • Start with Play mode to see the full algorithm flow
  • Use Step mode to examine each operation in detail
  • Try different inputs to understand edge cases and behavior
  • Use the undo feature to review previous steps
  • Adjust animation speed to match your learning pace

Note: Be careful when stepping forward and backward through animations - it's easy to confuse the next step with the previous step. When first learning an algorithm, consider only stepping forward to avoid confusion.
